Available Options

  • mode

    Purpose:Mode of operation
    Default: 'datebox'
    Options: 'datebox', 'slidebox', 'calbox', 'timebox', 'durationbox', 'flipbox', 'timeflipbox'

  • Themeing

    • Named Themes

    • theme

      Purpose:Theme for original input elemet
      Default: 'c'
      Modes: all

    • pickPageTheme

      Purpose:Theme for popup window or dialog
      Default: 'b'
      Modes: all

    • pickPageButtonTheme

      Purpose:Theme for widget buttons
      Default: 'a'
      Modes: all

    • pickPageInputTheme

      Purpose:Theme for widget input elemets
      Default: 'e'
      Modes: android, duration, time

    • pickPageSlideButtonTheme

      Purpose:Theme for non-centered slide buttons
      Default: 'd'
      Modes: slide

    • pickPageFlipButtonTheme

      Purpose:Theme for non-centered flip buttons
      Default: 'd'
      Modes: flip

    • pickPageHighButtonTheme

      Purpose:Theme for currently selected date button
      Default: 'e'
      Modes: calendar

    • pickPageTodayButtonTheme

      Purpose:Theme for current date's button
      Default: 'e'
      Modes: calendar, flip, slide

    • pickPageOHighButtonTheme

      Purpose:Theme for highlighted buttons (highDates)
      Default: 'e'
      Modes: calendar

    • pickPageODHighButtonTheme

      Purpose:Theme for highlighted buttons (highDays)
      Default: 'e'
      Modes: calendar

    • Boolean Display Options

    • noAnimation

      Purpose:Disable popup window animations
      Default: false
      Modes: all

    • centerWindow

      Purpose:Center the popup window in the viewport, rather than over the input element
      Default: false
      Modes: all

    • disableManualInput

      Purpose:Only allow setting the value with the pickers, disable typing the value.
      Default: false
      Modes: all

    • calTodayButton

      Purpose:Show a button to jump to "today"
      Default: false
      Modes: calendar

    • calWeekModeHighlight

      Purpose:Toggle all buttons on week mode
      Default: true
      Modes: calendar

    • calHighToday

      Purpose:Use theme highlight for calendar "today"
      Default: true
      Modes: calendar

    • calHighPicked

      Purpose:Use theme highlight for calendar "picked date"
      Default: true
      Modes: calendar

    • Other Display Options

    • zindex

      Purpose:Z-Index for popup
      Default: 500
      Modes: all

    • disabledDayColor

      Purpose:Text color for disabled days
      Default: '#888'
      Modes: calendar

  • Date Formats

    • Output Format

    • dateFormat

      Purpose:Format the returned date
      Default: 'YYYY-MM-DD' for dates
      Modes: all date 

      	YYYY: 4 Digit Year
	MM : 2 Digit Month (zero pad)
	mm : 1 or 2 Digit Month
	mmm : Text Month
	DD : 2 Digit Day (zero pad)
	dd : 1 or 2 Digit Day
	ddd : Day of week
	SS : Ordinal Suffix (4th, 3rd, 1st, etc)
	HH : 2 Digit Hour, 24 Hour Clock (zero pad)
	hh : 1 or 2 Digit Hour, 24 Hour Clock
	GG : 2 Digit Hour, 12 Hour Clock (zero pad)
	gg : 1 or 2 Digit Hour, 12 Hour Clock
	ii : 2 Digit Minutes (zero pad)
	ss : 2 Digit Seconds (zero pad)
	aa : Lower case meridiem letters
	AA : Upper case meridiem letters

    • headerFormat

      Purpose:Format the date header
      Default: 'ddd, mmm dd, YYYY'
      Modes: android, slide 

      	YYYY: 4 Digit Year
	MM : 2 Digit Month (zero pad)
	mm : 1 or 2 Digit Month
	mmm : Text Month
	DD : 2 Digit Day (zero pad)
	dd : 1 or 2 Digit Day
	ddd : Day of week
	SS : Ordinal Suffix (4th, 3rd, 1st, etc)
	HH : 2 Digit Hour, 24 Hour Clock (zero pad)
	hh : 1 or 2 Digit Hour, 24 Hour Clock
	GG : 2 Digit Hour, 12 Hour Clock (zero pad)
	gg : 1 or 2 Digit Hour, 12 Hour Clock
	ii : 2 Digit Minutes (zero pad)
	ss : 2 Digit Seconds (zero pad)
	aa : Lower case meridiem letters
	AA : Upper case meridiem letters

    • timeOutput

      Purpose:Format the returned time
      Default: 'GG:ii AA' or 'HH:ii' dependant on timeFormat
      Modes: timebox, timeflipbox 

      	YYYY: 4 Digit Year
	MM : 2 Digit Month (zero pad)
	mm : 1 or 2 Digit Month
	mmm : Text Month
	DD : 2 Digit Day (zero pad)
	dd : 1 or 2 Digit Day
	ddd : Day of week
	SS : Ordinal Suffix (4th, 3rd, 1st, etc)
	HH : 2 Digit Hour, 24 Hour Clock (zero pad)
	hh : 1 or 2 Digit Hour, 24 Hour Clock
	GG : 2 Digit Hour, 12 Hour Clock (zero pad)
	gg : 1 or 2 Digit Hour, 12 Hour Clock
	ii : 2 Digit Minutes (zero pad)
	ss : 2 Digit Seconds (zero pad)
	aa : Lower case meridiem letters
	AA : Upper case meridiem letters

    • durationShort

      Purpose:Use the shortest mode possible to display duration
      Default: true
      Modes: duration

    • durationNoDays

      Purpose:Do not break off days from duration, only h:i:s
      Default: false
      Modes: duration

    • Input Format

    • timeFormat

      Purpose:Set either a 12 or 24 hour clock
      Default: 24
      Modes: time

    • fieldsOrder

      Purpose:Define the order of the fields
      Default: android: ['m', 'd', 'y'], slide: ['y', 'm', 'd'], time: ['h', 'i', 'a'] ('a' only displays on 12 hour format)
      Modes: android, time, slide

      	y : Year
	m : Month
	d : Day of Month (date)
	h : Hour
	i : Minute
	a : Meridiem

    • durationOrder

      Purpose:Define the order of the fields
      Default: ['d', 'h', 'i', 's']
      Modes: duration

      	d : Days
	h : Hours
	i : Minutes
	s : Seconds

    • calShowDays

      Purpose:Boolean show day names in grid
      Default: true
      Modes: calendar

    • calShowOnlyMonth

      Purpose:Show *only* this month, do not fill in empty boxes
      Default: false
      Modes: calendar

    • calStartDay

      Purpose:Day of week to start calendar on (numeric - 0=Sun, 1=Mon,... 6=Sat)
      Default: 0
      Modes: calendar

    • Steppers

    • minuteStep

      Purpose:Number of minutes to step per button press in timebox mode
      Default: 1
      Modes: time

    • durationSteppers

      Purpose:An object of the amount to step each part of the duration
      Default: {'d':1, 'h':1, 'i':1, 's':1}
      Modes: duration

  • Date Defaults and Limits

    • Defaults

    • defaultDate

      Purpose:Default date when nothing entered into input box, array of [y,m,d] prefered, ISO accepted. For durationbox mode, this can be an integer of seconds instead.
      Default: false
      Modes: all

    • Limits

    • afterToday

      Purpose:Limit date to "today" or after
      Default: false
      Modes: android, calendar

    • beforeToday

      Purpose:Limit date to "today" or before
      Default: false
      Modes: android, calendar

    • notToday

      Purpose:Limit date to NOT "today"
      Default: false
      Modes: calendar

    • maxDays

      Purpose:Set the upper limit to this # of days after today
      Default: false
      Modes: android, calendar

    • minDays

      Purpose:Set the lower limit to this # of days before today
      Default: false
      Modes: android, calendar

    • maxYear

      Purpose:Maximum year allowed in controls
      Default: false
      Modes: android

    • minYear

      Purpose:Minimum year allowed in controls
      Default: false
      Modes: android

    • calWeekMode

      Purpose:Boolean, week mode on or off (choose only a specific DAY)
      Default: false
      Modes: calendar

    • calWeekModeFirstDay

      Purpose:Day of week to choose with week mode (numeric, 0-6, relative to calendar display)
      Default: false
      Modes: calendar

    • blackDays

      Purpose:An array of days of the week to blacklist
      Default: false
      Modes: calendar
      Example: [0,2] : Blacklist sunday and tuesday

    • blackDates

      Purpose:A simple array of individual dates to blacklist
      Default: false
      Modes: calendar
      Example: [ '2010-12-24', '2010-12-25' ] : Blacklist December 24th & 25th, 2010.

    • highDays

      Purpose:An array of days of the week to highlight
      Default: false
      Modes: calendar
      Example: [0,2] : Highlight sunday and tuesday

    • highDates

      Purpose:A simple array of individual dates to highlight with pickPageOHighTheme
      Default: false
      Modes: calendar
      Example: [ '2010-12-24', '2010-12-25' ] : Blacklist December 24th & 25th, 2010.

  • User Interface

    • useDialogForceTrue

      Purpose:Boolean *Always* use Dialog Window, regardless of screen size
      Default: false
      Modes: all

    • useDialogForceFalse

      Purpose:Boolean *Never* use Dialog Window, regardless of screen size
      Default: false
      Modes: all

    • useModal

      Purpose:Use modal styling (background fades out) on popup
      Default: false
      Modes: all

    • nestedBox

      Purpose:Use when datebox might be initilized somewhere other than where it is opened - i.e. in a simpleDialog
      Default: false
      Modes: all

    • noButtonFocusMode

      Purpose:Suppress button, open on element focus instead
      Default: false
      Modes: all

    • noButton

      Purpose:Suppress button
      Default: false
      Modes: all

    • noSetButton

      Purpose:Suppress the set button in the dialog
      Default: false
      Modes: android, slide, duration, time

    • useInline

      Purpose:Show datebox inline on page, no dialog or popup
      Default: false
      Modes: all

    • useInlineHideInput

      Purpose:When using inline mode, do not show the input box
      Default: false
      Modes: all

    • closeCallback

      Purpose:A function to run on DateBox close
      Default: false
      Modes: all

    • swipeEnabled

      Purpose:Swipe gestures active (Note: disableing in flip varients or slide will render them pretty much useless)
      Default: true
      Modes: all

  • Text Internationaliztion

    • Months and Days

    • daysOfWeek

      Purpose:An array of the days of the week
      Default: ['Sunday', 'Monday', ... 'Saturday']
      Modes: all

    • daysOfWeekShort

      Purpose:An array of the short names of the days of the week
      Default: ['Su', 'Mo', ... 'Sa']
      Modes: calendar, slide

    • monthsOfYear

      Purpose:An array of the months of the year
      Default: ['January', 'February', ... 'December']
      Modes: all

    • monthsOfYearShort

      Purpose:An array of the short names of the months of the year
      Default: ['Jan', 'Feb', ... 'Dec']
      Modes: all

    • durationLabel

      Purpose:An array of the parts of a duration
      Default: ['Days', 'Hours', 'Minutes', 'Seconds']
      Modes: duration

    • durationDays

      Purpose:An array of the single and plural name for 'Day'
      Default: ['Day', 'Days']
      Modes: duration

    • meridiemLetters

      Purpose:An array of meridiem notation
      Default: ['AM', 'PM']
      Modes: time

    • Buttons and Headers

    • setDateButtonLabel

      Purpose:The 'Set Date' button label
      Default: 'Set Date'
      Modes: android, slide

    • setDurationButtonLabel

      Purpose:The 'Set Duration' button label
      Default: 'Set Duration'
      Modes: duration

    • setTimeButtonLabel

      Purpose:The 'Set Time' button label
      Default: 'Set Time'
      Modes: time

    • calTodayButtonLabel

      Purpose:The 'Jump to Today' button label
      Default: 'Jump to Today'
      Modes: calendar

    • titleDateDialogLabel

      Purpose:Dialog label for date modes
      Default: 'Set Date'
      Modes: android, calendar, slide

    • titleTimeDialogLabel

      Purpose:Dialog label for time modes
      Default: 'Set Time'
      Modes: time

    • titleDialogLabel

      Purpose:Override for dialog label
      Default: false
      Modes: all
